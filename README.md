# Vrinda Store Sales Dashboard 2022

## Overview
This project is an interactive Excel dashboard created to analyze and visualize Vrinda Store sales data for the year 2022. The dashboard provides insights into sales performance, customer demographics, order status, sales channels, and state-wise sales distribution.

---

## Key Insights

- Women customers contributed more sales compared to men.
- Most orders were successfully delivered.
- Amazon, Myntra, and Flipkart were the top-performing sales channels.
- Maharashtra recorded the highest sales among all states.
- Adult women contributed the highest percentage of orders.

---

## Features

- Interactive Slicers for filtering:
  - Month
  - Category
  - Channel

- Visualizations Included:
  - Sales by Month
  - Sales Men vs Women
  - Order Status Analysis
  - Top 10 States by Sales
